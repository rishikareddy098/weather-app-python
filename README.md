## Description
A Python console application that retrieves real-time weather information using the OpenWeatherMap API.

## Features
- Enter city name
- Displays temperature,weather condition, and humidity
- Handles invalid city names, and network errors

## Technologies Used
- Python
- Requests
- JSON
- OpenWeatherMap API

## Installation
```bash
py -m pip install requests
```

## Run
```bash
py weather_app.py
